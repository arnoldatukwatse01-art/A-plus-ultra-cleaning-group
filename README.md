## Project Overview
This is a 4-page responsive business website for **A+plus Ultra-Cleaning Group**, a real cleaning services company based in Muyenga Bukasa, Kampala. 

The site showcases the company's services, products, and contact information using modern web technologies.

## Website Structure
1. **index.html** - Homepage with company intro and hero section
2. **services.html** - Detailed list of cleaning services offered with pricing
3. **products.html** - Cleaning products catalog with prices in UGX
4. **contact.html** - Contact form, phone, location, and business hours
5. **style.css** - Custom styles supplementing Bootstrap framework
